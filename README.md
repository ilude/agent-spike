# Agent Spike

AI agent spike for exploration and development.

## Development

This project uses:
- **Python 3.14**
- **uv** for package management
- **Typer** for CLI
- **pytest** for testing
- **devcontainer** for development environment

### Getting Started

Build and enter the devcontainer:

```bash
make build-dev
```

Run tests:

```bash
uv run pytest
```

Format code:

```bash
make format
```

Lint code:

```bash
make lint
```
